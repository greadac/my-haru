📱 앱 소개

나의 하루는 일정·일기·말씀묵상·건강을 한 곳에서 기록하는 개인 다이어리 앱입니다.
모든 데이터는 이 기기의 브라우저에만 저장되며, 외부 서버로 전송되지 않습니다.

🗓️ 달력 탭

• 달력에서 날짜를 탭하면 해당 날의 일정·일기·말씀 기록을 확인할 수 있습니다.
• ‹ › 버튼으로 월을 이동합니다.
• 날짜 아래 색상 점: 🟠 일정 / 🟢 일기 / 🟣 말씀
• 우측 하단 + 버튼으로 일정을 빠르게 추가합니다.

📋 일정 탭

• 전체 일정을 날짜순으로 조회합니다.
• 상단 필터 칩으로 카테고리별 필터링이 가능합니다.
• 카테고리: 💼 업무 / 🙂 개인 / 🤝 약속 / 🎉 기념일 / 📌 기타
• 일정을 탭하면 수정 또는 삭제할 수 있습니다.

📓 일기 탭

• 날짜별로 일기를 작성합니다.
• 제목·내용·기분을 기록할 수 있습니다.
• 기분: 😊 좋음 / 😐 보통 / 😔 나쁨 / 😤 화남 / 😴 피곤
• 우측 하단 ✏️ 버튼으로 새 일기를 작성합니다.

✝️ 말씀 탭

• 묵상 목록: 성경 말씀 묵상을 날짜별로 기록합니다. 읽은 범위·구절·묵상·기도를 적을 수 있습니다.
• 문장수집: 성경이나 책에서 마음에 닿은 문장을 모아둡니다.
• 통독표: 시작일·시작 위치·하루 분량을 설정하면 자동으로 통독 계획표를 생성합니다.
• 성경은 가톨릭 성경(73권) 기준입니다.

🥗 건강 탭

• 날짜 탭을 선택해 해당 날의 기록을 입력합니다.
• 기상 몸무게·식사(아침/점심/저녁)·물 섭취량·운동·메모를 기록합니다.
• 전날 대비 체중 변화가 배지로 표시됩니다.
• 📊 차트 보기로 몸무게 변화 추이를 그래프로 확인할 수 있습니다.
• 날짜 탭은 시작일부터 오늘까지 + 이후 7일이 자동 생성됩니다.

⚙️ 설정 탭

• 탭 표시 설정: 사용하지 않는 탭(일기·말씀·건강)을 숨기거나 다시 표시할 수 있습니다.
• 데이터 내보내기: 전체 데이터를 JSON 파일로 저장합니다 (백업용).
• 데이터 불러오기: 저장해 둔 JSON 백업 파일을 복원합니다.

💡 자주 묻는 질문

Q. 데이터가 사라졌어요.
브라우저 캐시/쿠키 삭제 시 localStorage 데이터도 함께 삭제됩니다.
설정 탭에서 정기적으로 백업하는 것을 권장합니다.

Q. 다른 기기에서도 쓸 수 있나요?
데이터 내보내기 후 다른 기기에서 불러오기를 사용하세요.

Q. 탭이 보이지 않아요.
설정 탭에서 탭 표시 설정을 확인해보세요.
